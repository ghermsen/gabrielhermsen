Logistic Dashboard with Power BI

  • Project's Files

Dashboard created to perform profit and loss analysis. In this Dashboard, I used public data from Siemens' annual reports, and I extracted the stock prices from Yahoo Finance to its development.

This Dashboard has the following features:

  • Profit and Loss analysis with Vertical and Horizontal Analysis
  • monitoring of the company's share price
  • scenario simulator

1 - Stock Price Monitor

(back to dashboard) | (list of measures)


1.1 - Absolute Change

(back to dashboard) | (list of measures) 

AbsoluteChange = [Last Close Price] - [Before Yesterday Close]

1.2 - Before Yesterday Close

(back to dashboard) | (list of measures) 

Before Yesterday Close = 
VAR vLastDate = LASTDATE(fStockPrice[Date])
VAR vBeforeDates = 
FILTER(
    ALL(
        fStockPrice[Date]), 
    fStockPrice[Date] < vLastDate
)
RETURN
CALCULATE(
    _Measures_Stock_Price[Price],
    LASTDATE(vBeforeDates)
)

1.3 - Last Close Price

(back to dashboard) | (list of measures) 

Last Close Price = 
CALCULATE(
    _Measures_Stock_Price[Price],
    LASTDATE(fStockPrice[Date]))

1.4 - Last High Price

(back to dashboard) | (list of measures) 

LastHighPrice = 
CALCULATE(
    MAX(fStockPrice[High]),
    LASTDATE(fStockPrice[Date])
)

1.5 - Last Low Price

(back to dashboard) | (list of measures) 

LastLowPrice = 
CALCULATE(
    MAX(fStockPrice[Low]),
    LASTDATE(fStockPrice[Date])
)

1.6 - Last Open Price

(back to dashboard) | (list of measures) 

LastOpenPrice = 
CALCULATE(
    MAX(fStockPrice[Open]),
    LASTDATE(fStockPrice[Date])
)

1.7 - Percent Change

(back to dashboard) | (list of measures) 

Change = 
VAR vAbsoluteChange = [Last Close Price] - [Before Yesterday Close]
VAR vPercentChange = DIVIDE([Last Close Price], [Before Yesterday Close])
VAR vDirection = IF(vAbsoluteChange > 0, "↑", "↓")
RETURN
FORMAT(vAbsoluteChange, "0.00") &
" (" &
FORMAT(vPercentChange - 1, "0.00%") &
")" &
vDirection

1.8 - Price

(back to dashboard) | (list of measures) 

Price = MAX(fStockPrice[Close])

1.9 - Table Periods

(back to dashboard) | (list of measures) 

fPeriods = 
UNION(
ADDCOLUMNS(DATESINPERIOD(dCalendar[Date], TODAY(),-5,DAY), "Period", "5 days", "Index", 1),
ADDCOLUMNS(DATESINPERIOD(dCalendar[Date], TODAY(),-1,MONTH), "Period", "1 month", "Index", 2),
ADDCOLUMNS(DATESINPERIOD(dCalendar[Date], TODAY(),-6,MONTH), "Period", "6 months", "Index", 3),
ADDCOLUMNS(DATESYTD(dCalendar[Date]), "Period", "YTD", "Index", 4),
ADDCOLUMNS(DATESINPERIOD(dCalendar[Date], TODAY(),-1,YEAR), "Period", "1 year", "Index", 5),
ADDCOLUMNS(DATESINPERIOD(dCalendar[Date], TODAY(),-5,YEAR), "Period", "5 years", "Index", 6),
ADDCOLUMNS(DATESINPERIOD(dCalendar[Date], TODAY(),-30,YEAR), "Period", "Max", "Index", 7)
)

2 - Profit and Loss Statement Matrix

(back to dashboard) | (list of measures)


2.1 - Cost of Sales

(back to dashboard) | (list of measures) 

Cost of Sales = 
CALCULATE(
    [Total Result] * 1000000,
    dDescription[ID Description] = 2
)

2.2 - Expenses

(back to dashboard) | (list of measures) 

Expenses = 
var vID4 = 
CALCULATE(
    [Total Result] * 1000000,
    dDescription[ID Description] = 4
)
var vID5 = 
CALCULATE(
    [Total Result] * 1000000,
    dDescription[ID Description] = 5
)
var vID7 = 
CALCULATE(
    [Total Result] * 1000000,
    dDescription[ID Description] = 7
)
var vID10 = 
CALCULATE(
    [Total Result] * 1000000,
    dDescription[ID Description] = 10
)
RETURN
vID4 + vID5 + vID7 + vID10

2.3 - Gross Profit

(back to dashboard) | (list of measures) 

Gross Profit = 
CALCULATE(
    [Revenue] + [Cost of Sales],
    dDescription[ID Description] = 3
)

2.4 - Gross Profit Margin

(back to dashboard) | (list of measures) 

Gross Profit Margin = 
VAR vGrossProfit = 
CALCULATE(
    [Statement],
    dDescription[ID Description] = 3
)
VAR vRevenue = 
CALCULATE(
    [Statement],
    dDescription[ID Description] = 1
)
VAR vGrossProfitMargin =
DIVIDE(
    vGrossProfit, vRevenue
)
Return
vGrossProfitMargin

2.5 - Horizontal Analysis

(back to dashboard) | (list of measures) 

HA Statement = 
var vLastYear = CALCULATE([Statement], SAMEPERIODLASTYEAR(dCalendar[Date]))
var vHR =
DIVIDE(
    [Statement] - vLastYear,
    ABS(vLastYear)
)
Return
vHR

)

2.7 - Net Income

(back to dashboard) | (list of measures)

    dDescription[ID Description] = 16
)

2.8 - Revenue

(back to dashboard) | (list of measures) 

Revenue = 
CALCULATE(
    [Total Result] * 1000000,
    dDescription[ID Description] = 1
)

2.9 - Sum of Cost and Expenses

(back to dashboard) | (list of measures) 

Sum Cost and Expenses = [Cost of Sales] + [Expenses]

2.10 - Vertical Analysis

(back to dashboard) | (list of measures) 

VA Statement = 
var FixedResult =
CALCULATE(
    [Statement],
    ALL(dDescription),
    dDescription[ID Description] = 1
)
Return
DIVIDE(
    [Statement],
    FixedResult
)
